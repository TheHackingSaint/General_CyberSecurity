# **Tools in CyberSecurity**

---

**Splunk : a platform for collecting, indexing, searching, and visualizing machine‑generated data.**

Splunk is an enterprise‑grade data analytics and SIEM platform used to ingest logs from servers, applications, network devices, and security tools, then index and correlate them for monitoring, alerting, and threat detection. It provides powerful search capabilities through the Splunk Search Processing Language (SPL), enabling analysts to investigate incidents, track system performance, visualize trends, and detect anomalies in real time. Its role in cybersecurity includes log forensics, incident response, and compliance monitoring.

[Splunk](https://www.splunk.com/)
