# **Tools in CyberSecurity**

---

**Splunk : a platform for collecting, indexing, searching, and visualizing machine‑generated data.**

Splunk is an enterprise‑grade data analytics and SIEM platform used to ingest logs from servers, applications, network devices, and security tools, then index and correlate them for monitoring, alerting, and threat detection. It provides powerful search capabilities through the Splunk Search Processing Language (SPL), enabling analysts to investigate incidents, track system performance, visualize trends, and detect anomalies in real time. Its role in cybersecurity includes log forensics, incident response, and compliance monitoring.

[Splunk](https://www.splunk.com/)

---

**Burp Suite: a web application security testing platform used to intercept, analyze, and manipulate HTTP/S traffic.**

Burp Suite is a comprehensive toolkit for web penetration testing that intercepts browser traffic, allowing you to inspect requests, modify them, and replay them to test how an application responds. It includes components like Proxy for traffic interception, Repeater for manual testing, Intruder for automated attacks, Decoder and Comparer for data analysis, and Scanner in the Professional edition for automated vulnerability detection. It’s central to workflows involving parameter tampering, authentication testing, input manipulation, and overall web security assessment.

`pre-installed in Kali`
